# Interface Gmail

Interface do Gmail com React Native, Tailwind e Expo Router

# Para iniciar:

```bash
 npx create-expo-app --template
```

-> Navigation (TypeScript)

Criar src/app/(tabs)

```bash
 npm install nativewind@^4.0.1 react-native-reanimated tailwindcss
```

```bash
 npx tailwindcss init
```
